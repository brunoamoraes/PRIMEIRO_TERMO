## 📚 Disciplinas e Focos de Estudo

### 1. 🌐 Arquitetura IoT
*   **Hardware:** Desenvolvimento com **Arduino** (Sensores e Atuadores).
*   **Linguagens:** Firmware em **C++** e integração de dados com **Python**.
*   **Conceito:** Comunicação entre dispositivos e arquitetura de borda (*Edge Computing*).

### 2. 🖥️ Sistemas Operacionais
*   **Ecossistemas:** Administração e fundamentos de **Windows, Linux e iOS**.
*   **Kernel:** Estudo de gerenciamento de processos, memória e sistemas de arquivos.
*   **Terminal:** Operação via CLI (Prompt de Comando, PowerShell e Bash).

### 3. 📋 Engenharia de Requisitos
*   **Elicitação:** Técnicas de **Entrevistas, Brainstorming e Prototipagem**.
*   **Modelagem:** Documentação de Requisitos Funcionais (RF) e Não Funcionais (RNF).
*   **Diagramação:** Elaboração de diagramas UML e fluxos de processos.

### 4. 🐍 Lógica de Programação
*   **Desenvolvimento:** Algoritmos estruturados utilizando **Python**.
*   **Qualidade:** Aplicação de princípios de **Clean Code** para código legível e escalável.
*   **Versionamento:** Fluxo de trabalho profissional com **Git & GitHub**.
